# 🚀 Python FastAPI CRUD API

A production-ready asynchronous backend API built with **Python**, **FastAPI**, and **SQLite**. This project showcases a full CRUD (Create, Read, Update, Delete) workflow for session management, designed with clean architecture principles.

---

## 🛠️ Technologies Used

* **Python 3**
* **FastAPI** (Modern, fast web framework for building APIs)
* **Uvicorn** (Lightning-fast ASGI server implementation)
* **SQLite 3** (Lightweight relational database)

---

## ⚙️ Features & Endpoints

The API automatically generates interactive documentation (Swagger UI) at `/docs`.

* `GET /` - Fetch all saved data/sessions from the database.
* `POST /` - Insert new records dynamically into the database.
* `DELETE /deletar-sessao/{id}` - Delete specific entries using unique path parameters.
